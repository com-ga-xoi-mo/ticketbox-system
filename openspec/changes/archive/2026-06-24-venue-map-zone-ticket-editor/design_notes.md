# Implementation Notes for Venue Map & Ticket Mapping Editor

Based on the Stitch screen 'TicketBox - Venue Map & Ticket Mapping Editor':
- Left side: SVGs rendered centrally, showing the seating map.
- Top bar: Context of the current concert, "Venue Maps", with back button and publish/status controls.
- Right side: Tabs for "SEATING ZONES", "TICKET TYPES", and "MAPPING".
- Inputs use standard dark mode styling.
- All technical identifiers (e.g. svgElementIds) use mono fonts.
- Pricing in VND.
